# Gyansar

Kotlin Multiplatform (Android + iOS) app.

## Build

### Android

```bash
./gradlew :composeApp:assembleDebug
```

### iOS

Open `iosApp/iosApp.xcodeproj` in Xcode and run.

## Notes

This project was initially bootstrapped from JetBrains KMP templates and has been customized for this repo.
